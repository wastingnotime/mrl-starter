# Cross-Repository Findings Guidance

This repository follows the global `cross-repo-findings` skill.

Keep the local guidance short:

- write the finding under `work/findings/<date>-<short-name>/finding.md`
- open a GitHub issue in this repository
- post a note to Discord `#findings` (`1502715067186024628`)
- close the issue when the finding is resolved
- send a completion message to `#findings` when that issue closes
- escalate multi-repository impact to management

Do not implement another repository's fix locally.
